CSS

CSS (Cascading Style Sheets) é a linguagem usada para definir a aparência e o layout das páginas feitas com HTML, controlando cores, fontes, espaçamentos, tamanhos e posicionamento dos elementos; em resumo, enquanto o HTML estrututura o conteúdo, o CSS deixa tudo visualmente organizado e bonito.
